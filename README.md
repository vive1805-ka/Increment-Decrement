 ⚛️ React Counter App

A simple and interactive **React Counter Application** built using **useState**, **useEffect**, and **Props**. This project demonstrates React state management, component communication, event handling, and side effects through an increment and decrement counter.

## 🚀 Live Demo

🔗 https://increment-decrement-sand.vercel.app

---

## ✨ Features

• ➕ Increment Counter  
• ➖ Decrement Counter  
• ⚛️ State management using **useState**  
• 🔄 Side effects using **useEffect**  
• 🔗 Parent-to-child communication using **Props**  
• ⏳ Automatically resets to **0** after 2 seconds when the value becomes negative  
• 📱 Responsive and simple user interface
